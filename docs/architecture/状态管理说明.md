## Redux Store 架构

### Store 结构

```typescript
RootState = {
  auth: AuthState,         // 认证状态
  ui: UiState,             // UI 状态
  exercise: ExerciseState, // 练习题状态
  session: SessionState,   // 学习会话状态
  analytics: AnalyticsState, // 学习分析状态
}
```

### Store 配置

**文件位置**：`src/store/index.ts`

```typescript
import { configureStore } from '@reduxjs/toolkit';
import authReducer from './slices/authSlice';
import uiReducer from './slices/uiSlice';
import exerciseReducer from './slices/exerciseSlice';
import sessionReducer from './slices/sessionSlice';
import analyticsReducer from './slices/analyticsSlice';

export const store = configureStore({
  reducer: {
    auth: authReducer,
    ui: uiReducer,
    exercise: exerciseReducer,
    session: sessionReducer,
    analytics: analyticsReducer,
  },
});

export type RootState = ReturnType<typeof store.getState>;
export type AppDispatch = typeof store.dispatch;
```

## Auth Slice（认证状态）

**文件位置**：`src/store/slices/authSlice.ts`

### 状态定义

```typescript
interface AuthState {
  token: string | null;
  user: {
    id: string;
    name: string;
    email?: string;
    role: 'student' | 'teacher';
    avatar?: string;
  } | null;
  isAuthenticated: boolean;
  loadingState: LoadingState;
  error: string | null;
}
```

### Actions

| Action | 参数 | 说明 |
|--------|------|------|
| `setCredentials` | `{ token: string, user: User }` | 设置登录凭证 |
| `logout` | 无 | 登出（清除状态和 localStorage） |
| `updateUser` | `Partial<User>` | 更新用户信息 |
| `setLoadingState` | `LoadingState` | 设置加载状态 |
| `setError` | `string` | 设置错误信息 |
| `clearError` | 无 | 清除错误 |
| `refreshToken` | `string` | 刷新 Token |

### Selectors

```typescript
selectCurrentUser(state)       // 获取当前用户
selectIsAuthenticated(state)   // 是否已登录
selectAuthToken(state)          // 获取 Token
selectAuthLoadingState(state)   // 获取加载状态
selectAuthError(state)          // 获取错误信息
selectIsAuthLoading(state)      // 是否正在加载
```

### 使用示例

```typescript
import { useDispatch, useSelector } from 'react-redux';
import { setCredentials, logout, selectCurrentUser } from '@/store/slices/authSlice';

function LoginComponent() {
  const dispatch = useDispatch();
  const user = useSelector(selectCurrentUser);

  const handleLogin = async (credentials) => {
    const response = await authService.login(credentials);
    dispatch(setCredentials({
      token: response.token,
      user: response.user,
    }));
  };

  const handleLogout = () => {
    dispatch(logout());
  };

  return (
    <div>
      {user ? (
        <div>
          <p>欢迎，{user.name}</p>
          <button onClick={handleLogout}>登出</button>
        </div>
      ) : (
        <button onClick={handleLogin}>登录</button>
      )}
    </div>
  );
}
```

### 持久化

- Token 存储在 `localStorage` 中（key: `auth_token`）
- 页面刷新时从 `localStorage` 恢复认证状态
- 登出时自动清除 `localStorage`

### 使用示例

```typescript
import { useDispatch, useSelector } from 'react-redux';
import {
  setCurrentExercise,
  setCurrentAnswer,
  setFeedback,
  selectCurrentExercise,
  selectCurrentAnswer,
  selectIsExerciseSubmitting,
} from '@/store/slices/exerciseSlice';

function ExercisePage() {
  const dispatch = useDispatch();
  const exercise = useSelector(selectCurrentExercise);
  const answer = useSelector(selectCurrentAnswer);
  const isSubmitting = useSelector(selectIsExerciseSubmitting);

  const handleSubmit = async () => {
    if (!exercise) return;

    dispatch(setSubmitState('loading'));
    try {
      const response = await exerciseService.submitAnswer({
        exerciseId: exercise.id,
        answer,
      });
      dispatch(setFeedback({
        correct: response.correct,
        explanation: response.feedback,
      }));
      dispatch(setSubmitState('success'));
    } catch (error) {
      dispatch(setError(error.message));
    }
  };

  return (
    <div>
      {exercise && (
        <>
          <h2>{exercise.title}</h2>
          <div>{exercise.content}</div>
          <input
            value={answer}
            onChange={(e) => dispatch(setCurrentAnswer(e.target.value))}
          />
          <button onClick={handleSubmit} disabled={isSubmitting}>
            {isSubmitting ? '提交中...' : '提交答案'}
          </button>
        </>
      )}
    </div>
  );
}
```

### 使用示例

```typescript
import { useDispatch, useSelector } from 'react-redux';
import {
  setInputContent,
  addMessage,
  setIsTyping,
  selectMessages,
  selectInputContent,
  selectIsTyping,
} from '@/store/slices/sessionSlice';

function ChatInterface() {
  const dispatch = useDispatch();
  const messages = useSelector(selectMessages);
  const inputContent = useSelector(selectInputContent);
  const isTyping = useSelector(selectIsTyping);

  const handleSend = async () => {
    if (!inputContent.trim()) return;

    // 添加用户消息
    dispatch(addMessage({
      id: Date.now().toString(),
      role: 'user',
      content: inputContent,
      timestamp: new Date().toISOString(),
    }));

    dispatch(setInputContent(''));
    dispatch(setIsTyping(true));

    try {
      const response = await sessionService.sendMessage(sessionId, inputContent);
      dispatch(addMessage(response));
    } finally {
      dispatch(setIsTyping(false));
    }
  };

  return (
    <div>
      <div className="messages">
        {messages.map((msg) => (
          <div key={msg.id} className={msg.role}>
            {msg.content}
          </div>
        ))}
        {isTyping && <div>AI 正在输入...</div>}
      </div>
      <input
        value={inputContent}
        onChange={(e) => dispatch(setInputContent(e.target.value))}
        onKeyPress={(e) => e.key === 'Enter' && handleSend()}
      />
      <button onClick={handleSend}>发送</button>
    </div>
  );
}
```

### 2. 异步操作

#### 方式一：在组件中处理（当前方式）

```typescript
function MyComponent() {
  const dispatch = useDispatch();

  const handleLogin = async () => {
    dispatch(setLoadingState('loading'));
    try {
      const response = await authService.login(credentials);
      dispatch(setCredentials(response));
      dispatch(setLoadingState('success'));
    } catch (error) {
      dispatch(setError(error.message));
    }
  };
}
```

#### 方式二：使用 Redux Thunk（推荐用于复杂逻辑）

```typescript
// src/store/slices/authSlice.ts
import { createAsyncThunk } from '@reduxjs/toolkit';

export const loginAsync = createAsyncThunk(
  'auth/login',
  async (credentials: LoginCredentials) => {
    const response = await authService.login(credentials);
    return response;
  }
);

// 在 slice 中添加 extraReducers
const authSlice = createSlice({
  name: 'auth',
  initialState,
  reducers: { /* ... */ },
  extraReducers: (builder) => {
    builder
      .addCase(loginAsync.pending, (state) => {
        state.loadingState = 'loading';
      })
      .addCase(loginAsync.fulfilled, (state, action) => {
        state.token = action.payload.token;
        state.user = action.payload.user;
        state.isAuthenticated = true;
        state.loadingState = 'success';
      })
      .addCase(loginAsync.rejected, (state, action) => {
        state.error = action.error.message || '登录失败';
        state.loadingState = 'error';
      });
  },
});

// 在组件中使用
function MyComponent() {
  const dispatch = useAppDispatch();

  const handleLogin = () => {
    dispatch(loginAsync(credentials));
  };
}
```

### 3. 状态持久化

#### 当前实现（手动持久化）

```typescript
// authSlice.ts
const authSlice = createSlice({
  name: 'auth',
  initialState: {
    token: localStorage.getItem('auth_token'), // 从 localStorage 恢复
    // ...
  },
  reducers: {
    setCredentials(state, action) {
      state.token = action.payload.token;
      localStorage.setItem('auth_token', action.payload.token); // 保存到 localStorage
    },
    logout(state) {
      state.token = null;
      localStorage.removeItem('auth_token'); // 清除 localStorage
    },
  },
});
```

#### 使用 redux-persist（可选）

如果需要持久化更多状态，可以使用 `redux-persist` 库：

```bash
npm install redux-persist
```

```typescript
// src/store/index.ts
import { persistStore, persistReducer } from 'redux-persist';
import storage from 'redux-persist/lib/storage';

const persistConfig = {
  key: 'root',
  storage,

export const store = configureStore({
  reducer: persistedReducer,

export const persistor = persistStore(store);
```

### 5. 最佳实践

#### ✅ 推荐做法

1. **使用 Selectors**：通过 selector 访问状态，便于重构和优化
   ```typescript
   const user = useSelector(selectCurrentUser); // ✅
   ```

2. **避免直接访问 state**：
   ```typescript
   const user = useSelector((state) => state.auth.user); // ❌ 不推荐
   ```

3. **使用类型化的 Hooks**：
   ```typescript
   const dispatch = useAppDispatch(); // ✅
   const user = useAppSelector(selectCurrentUser); // ✅
   ```

4. **保持 Slice 职责单一**：每个 Slice 只管理相关的状态

5. **使用 Immer**：Redux Toolkit 内置 Immer，可以直接修改 state
   ```typescript
   state.user.name = 'New Name'; // ✅ Redux Toolkit 会自动处理不可变性
   ```

6. **错误处理**：统一使用 `error` 字段存储错误信息

7. **加载状态**：使用 `LoadingState` 类型统一管理加载状态

#### ❌ 避免的做法

1. 不要在 reducer 中执行异步操作
2. 不要在 reducer 中修改原始 state（Redux Toolkit 除外）
3. 不要在 selector 中执行复杂计算（使用 reselect 库优化）
4. 不要在多个 Slice 中存储重复数据

## 状态流转图

### 认证流程

```
用户点击登录
  ↓
dispatch(setLoadingState('loading'))
  ↓
调用 authService.login()
  ↓
成功 → dispatch(setCredentials({ token, user }))
  ↓
localStorage.setItem('auth_token', token)
  ↓
isAuthenticated = true
  ↓
路由跳转到主页
```

### 练习题提交流程

```
用户输入答案
  ↓
dispatch(setCurrentAnswer(answer))
  ↓
用户点击提交
  ↓
dispatch(setSubmitState('loading'))
  ↓
调用 exerciseService.submitAnswer()
  ↓
成功 → dispatch(setFeedback(feedback))
  ↓
自动添加到 history
  ↓
显示反馈信息
```
