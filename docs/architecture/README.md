## 🏗️ 架构概览

### 系统架构

```
┌─────────────────────────────────────────────────────────┐
│                      前端层 (React)                      │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌─────────┐ │
│  │ 学生端   │  │ 教师端   │  │ 管理员端 │  │ 公共组件│ │
│  └──────────┘  └──────────┘  └──────────┘  └─────────┘ │
│         │              │              │           │      │
│         └──────────────┴──────────────┴───────────┘      │
│                        │                                 │
│                   Redux Store                            │
└─────────────────────────┬───────────────────────────────┘
                          │ HTTP/WebSocket
┌─────────────────────────┴───────────────────────────────┐
│                    后端层 (FastAPI)                      │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌─────────┐ │
│  │ API 路由 │  │ 业务逻辑 │  │ AI 智能体│  │ 数据访问│ │
│  └──────────┘  └──────────┘  └──────────┘  └─────────┘ │
└─────────────────────────┬───────────────────────────────┘
                          │
┌─────────────────────────┴───────────────────────────────┐
│                      数据层                              │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌─────────┐ │
│  │PostgreSQL│  │  Redis   │  │ Vector DB│  │ 文件存储│ │
│  └──────────┘  └──────────┘  └──────────┘  └─────────┘ │
└─────────────────────────────────────────────────────────┘
```

### 数据流

1. **用户请求** → 前端组件
2. **Dispatch Action** → Redux Store
3. **API 调用** → 后端服务
4. **业务处理** → 数据库操作
5. **返回响应** → 更新 Store
6. **UI 更新** → 用户界面

### AI 工作流

```
用户输入 → Router Agent → 专业 Agent → Verifier → 响应输出
              ↓              ↓           ↓
           路由决策      执行任务    质量检查
```

## 📊 技术选型

### 前端技术栈
- **框架**: React 18 + TypeScript
- **状态管理**: Redux Toolkit
- **UI 组件**: Ant Design
- **构建工具**: Vite
- **图表库**: ECharts, G6

### 后端技术栈
- **框架**: FastAPI
- **ORM**: SQLAlchemy
- **AI 框架**: LangGraph, LangChain
- **任务队列**: Celery
- **API 文档**: OpenAPI (Swagger)

### 数据存储
- **关系数据库**: PostgreSQL 14+
- **缓存**: Redis 7+
- **向量数据库**: VectorChord (pgvector)
- **文件存储**: 本地文件系统 / OSS
