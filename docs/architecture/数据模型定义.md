interface ApiResponse<T = unknown> {

## 用户相关

### User（用户基础信息）
```typescript
interface User {
  id: string;
  username: string;
  email: string;
  role: UserRole;
  avatar?: string;
  createdAt: string;
  updatedAt: string;
}
```

**字段说明**：
- `id` - 用户唯一标识
- `username` - 用户名
- `email` - 邮箱
- `role` - 用户角色（student/teacher/admin）
- `avatar` - 头像 URL（可选）
- `createdAt` - 创建时间（ISO 8601 格式）
- `updatedAt` - 更新时间（ISO 8601 格式）

## 练习题

### Exercise（练习题）
```typescript
interface Exercise {
  id: string;
  title: string;
  content: string;
  difficulty: DifficultyLevel;
  knowledgeNodeIds: string[];
  solution?: string;
  hints?: string[];
  createdAt: string;
  updatedAt: string;
}
```

**字段说明**：
- `id` - 练习题唯一标识
- `title` - 题目标题
- `content` - 题目内容（LaTeX 格式）
- `difficulty` - 难度等级（easy/medium/hard）
- `knowledgeNodeIds` - 关联的知识节点 ID 列表
- `solution` - 标准答案（可选，学生端可能不返回）
- `hints` - 提示列表（可选）
- `createdAt` - 创建时间
- `updatedAt` - 更新时间

**前端使用示例**：
```typescript
// 渲染 LaTeX 公式
import katex from 'katex';
const html = katex.renderToString(exercise.content, { throwOnError: false });
```

interface ExerciseCardProps {

### 在 API 服务中使用类型
```typescript
import type { Exercise, PaginationResponse } from '@/types';
import { apiClient } from '@/libs/http/apiClient';

export const exerciseService = {
  async fetchExercises(params: {
    page: number;
    pageSize: number;
    difficulty?: DifficultyLevel;
  }): Promise<PaginationResponse<Exercise>> {
    const response = await apiClient.get('/exercise/list', { params });
    return response.data.data;
  },
};
```

### 在 Redux Slice 中使用类型
```typescript
import type { Exercise, LoadingState } from '@/types';
import { createSlice, type PayloadAction } from '@reduxjs/toolkit';

interface ExerciseState {
  currentExercise: Exercise | null;
  loadingState: LoadingState;
}

const initialState: ExerciseState = {
  currentExercise: null,
  loadingState: 'idle',
};

const exerciseSlice = createSlice({
  name: 'exercise',
  initialState,
  reducers: {
    setCurrentExercise(state, action: PayloadAction<Exercise>) {
      state.currentExercise = action.payload;
    },
  },
});
```

  id: string;

## 注意事项

1. **日期时间格式**：所有日期时间字段使用 ISO 8601 格式字符串（`YYYY-MM-DDTHH:mm:ssZ`）
2. **可选字段**：使用 `?` 标记可选字段，后端可能不返回这些字段
3. **LaTeX 格式**：`content`、`solution` 等字段包含 LaTeX 公式，需使用 KaTeX 渲染
4. **ID 类型**：所有 ID 字段使用 `string` 类型（后端可能使用 UUID 或其他格式）
5. **枚举类型**：使用 TypeScript 字面量类型（如 `'easy' | 'medium' | 'hard'`）而非枚举
